Política de Privacidade

Compromisso com a proteção de dados em aplicativos Flutter

Coleta de Dados

Nossos aplicativos Flutter coletam apenas os dados necessários para o funcionamento adequado do serviço:

  • Dados de Autenticação: Email e informações básicas de perfil para login seguro
  • Dados de Uso: Informações sobre funcionalidades utilizadas para melhorar a experiência
  • Dados Técnicos: Informações do dispositivo para otimização de performance
  • Notificações Push: Token para envio de notificações importantes

Armazenamento e Segurança

Utilizamos tecnologias seguras para proteger seus dados:

  • Firebase: Serviços Google com criptografia de ponta a ponta
  • Local Storage: Dados sensíveis armazenados localmente no dispositivo
  • Criptografia: Todos os dados transmitidos são criptografados
  • Backup Seguro: Backups automáticos com proteção adicional

Compartilhamento de Dados

Seus dados são protegidos e não são compartilhados com terceiros, exceto:

  • Quando exigido por lei ou ordem judicial
  • Para prestadores de serviços essenciais (Firebase, Google Cloud)
  • Com seu consentimento explícito

Seus Direitos

Você tem direito a:

  • Acesso: Solicitar cópia dos dados coletados
  • Correção: Atualizar informações incorretas
  • Exclusão: Solicitar remoção de dados pessoais
  • Portabilidade: Transferir dados para outro serviço
  • Revogação: Cancelar consentimento a qualquer momento

Cookies e Rastreamento

Nossos aplicativos podem utilizar:

  • Cookies Essenciais: Para funcionamento básico do app
  • Analytics: Firebase Analytics para melhorias de performance
  • Crashlytics: Relatórios de erro para estabilidade
  • Sem Rastreamento: Não coletamos dados para publicidade

Retenção de Dados

Mantemos seus dados apenas pelo tempo necessário:

  • Dados Ativos: Enquanto sua conta estiver ativa
  • Dados Inativos: Até 12 meses após último acesso
  • Backups: Até 30 dias após exclusão da conta
  • Logs Técnicos: Até 90 dias para segurança

Conformidade Legal

Nossos aplicativos seguem as principais regulamentações:

  • LGPD: Lei Geral de Proteção de Dados (Brasil)
  • GDPR: Regulamento Geral de Proteção de Dados (UE)
  • App Store: Diretrizes de Privacidade da Apple
  • Google Play: Políticas de Privacidade do Google
